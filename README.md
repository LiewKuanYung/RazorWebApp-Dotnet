# RazorWebApp-Dotnet
A simple razor web application built using C# on top of .NET 5

## Run app (VS Code)
1. Download the code
2. Open this workspace in VS code
3. Make sure you download C# extension and dotnet SDK
4. Open terminal run `dotnet watch run`
